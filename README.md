# 教學駕駛艙總覽

這是可直接上傳 GitHub Pages 的精簡版靜態網站，用來統一管理多個教學駕駛艙。

## 內容

- `index.html`：駕駛艙總入口
- `unit1/`：Unit 1 教學駕駛艙
- `unit1/assets/slides/`：簡報輪播圖片
- `unit1/assets/infographics/`：資訊圖表圖片

## 發布方式

將本資料夾內容放到 GitHub repository 的根目錄，或設定 GitHub Pages 發布來源指向此資料夾內容。

不要把原始 PDF、PPTX、`output/preview_pages/` 或 `output/artifact_workspace/` 一起放進 repository。
